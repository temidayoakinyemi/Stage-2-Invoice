# Invoice Management App

A responsive Invoice Management Application built with React for the HNG Frontend Stage 2 Task.

## 🚀 Features

- Create invoices
- View invoice list
- View invoice details
- Edit invoices
- Delete invoices
- Save invoice data with LocalStorage
- Filter invoices by status
- Mark invoices as Paid / Pending / Draft
- Dark mode toggle
- Responsive design for mobile, tablet, desktop

## 🛠 Tech Stack

- React JS
- React Router DOM
- CSS3
- LocalStorage

## 📂 Project Structure
